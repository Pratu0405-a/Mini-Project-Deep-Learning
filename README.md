# 🖼️ Image Processing for Object Highlighting using OpenCV

## 📌 Overview

This project demonstrates how basic image processing techniques can be used to highlight specific objects in an image using Python and OpenCV.

The system identifies a region of interest using thresholding and highlights it with color while keeping the rest of the image unchanged.



## 🎯 Objectives

* Learn fundamental image processing techniques
* Convert images to grayscale
* Apply thresholding for object detection
* Perform masking and selective coloring
* Visualize results using Matplotlib
* Save processed images



## ❓ Problem Statement

To develop a system that processes an image and highlights a specific region using grayscale conversion, thresholding, and masking.

---

## 🛠️ Technologies Used

* Python
* OpenCV
* NumPy
* Matplotlib
* Google Colab / Jupyter Notebook



## 📷 Output

* Displays original and processed image side by side
* Highlights object in green color
* Saves output as `output.jpg`
