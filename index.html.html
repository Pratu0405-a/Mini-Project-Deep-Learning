◆ Overview
What This Project Does

This project applies fundamental image processing techniques to identify and highlight specific regions within a digital image using Python and OpenCV. It demonstrates how classical computer vision methods can achieve precise object segmentation without the complexity of deep learning.

🌞
Grayscale Conversion
Reduces computational complexity by eliminating color data while preserving intensity variations crucial for detection.
◐
Binary Thresholding
Separates foreground objects from background by classifying each pixel based on its intensity value against a threshold.
🎨
Selective Masking
Applies color only to the detected object region, leaving the background completely unchanged and intact.
📊
Side-by-Side Visualization
Uses Matplotlib to display original and processed images together for clear before-and-after comparison.

◆ Methodology
How It Works

The system uses three classical image processing techniques working in sequence to achieve accurate object-only coloring.

01
Grayscale Conversion
The input color image is converted to grayscale using the luminosity formula: Gray = 0.299R + 0.587G + 0.114B. This reduces a 3-channel image to a single channel, simplifying further processing.
02
Binary Thresholding
A global threshold is applied to the grayscale image. Pixels below the threshold (dark object regions) become black in the mask; pixels above (white background) become white — creating a clean binary mask.
03
Mask Inversion & Coloring
The binary mask is inverted to isolate the object. The chosen color is applied only where the inverted mask is active (the object pixels), while background pixels remain untouched from the original image.
04
Result Visualization
The original and processed images are displayed side-by-side using Matplotlib for comparison. The final output is saved as a JPEG file for further use in analysis or documentation.

◆ Implementation
Core Python Code

Built with Python in Google Colab using OpenCV, NumPy, and Matplotlib. The complete implementation is concise and efficient.

object_highlight.py
import cv2
import numpy as np
import matplotlib.pyplot as plt
from google.colab import files

# Upload image via Colab interface
uploaded = files.upload()
image_name = list(uploaded.keys())[0]

# Read and convert color spaces
img = cv2.imread(image_name)
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

# Apply threshold — creates binary mask
_, mask = cv2.threshold(gray, 120, 255, cv2.THRESH_BINARY)

# Invert mask to target dark object pixels
mask_inv = cv2.bitwise_not(mask)

# Copy original and colorize only the object region
colorized = img.copy()
colorized[mask_inv == 255] = [0, 0, 255]  # Red in BGR

# Display side-by-side comparison
plt.figure(figsize=(10, 5))
plt.subplot(1, 2, 1); plt.title("Original")
plt.imshow(cv2.cvtColor(img, cv2.COLOR_BGR2RGB))
plt.axis("off")
plt.subplot(1, 2, 2); plt.title("Object Highlighted")
plt.imshow(cv2.cvtColor(colorized, cv2.COLOR_BGR2RGB))
plt.axis("off")
plt.show()

# Save output
cv2.imwrite("output.jpg", colorized)
print("Saved as output.jpg ✅")
